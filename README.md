# Rambu Jalan TK

Iseng-iseng bikin web interaktif buat belajar rambu lalu lintas, ditujukan untuk anak TK.

## Fitur

- Belajar rambu jalan dengan gambar dan penjelasan
- Kuis pilihan ganda
- Pasangkan kartu (matching game)
- Sistem bintang ⭐

## Cara Pakai

Buka `index.html` di browser, langsung jalan tanpa perlu install apapun.
